# Sumit Mishra — Portfolio

Glassmorphic bento-grid portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run sync` — refresh GitHub projects in `src/data/auto-content.json`
- `npm run lint` — ESLint

GitHub Actions syncs projects daily and on pushes to `main` that touch the sync script.
